# Notes

## Offensive Security
- Finds vulnerabilities by simulating attacks.
- Helps organizations improve security.
- Requires proper authorization.

## Defensive Security
- Protects systems, networks, and data.
- Detects and responds to cyber threats.
- Uses monitoring and security tools.

## Cybersecurity Career Paths
- Penetration Tester
- Security Analyst (SOC)
- Incident Responder
- Security Engineer
- Malware Analyst
- Digital Forensics
- Cloud Security Engineer
